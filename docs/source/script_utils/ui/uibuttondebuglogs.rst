UIButtonDebugLogs
==========

**NAMESPACE:**
   `UnityUtils.ScriptUtils.UI`
   
Usage
-------------
Add a 'UIButtonDebugLogs' Script to a UI button and edit the Booleans to your needs, will output a different Debug.Log for hover, hover exit, and click.
