UIButtonSpawnParticles
==========

**NAMESPACE:**
   `UnityUtils.ScriptUtils.UI`
   
Usage
-------------
Add a 'UIButtonSpawnParticles' Script to a UI button it will spawn particles on hover, exit, or click depending on the particle systems prefabs you added.
