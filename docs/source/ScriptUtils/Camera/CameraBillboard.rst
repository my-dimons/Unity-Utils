CameraBillboard
================

**NAMESPACE:**
   `UnityUtils.ScriptUtils.Cameras`
     
The **CameraBillboard** is used to easily have an object rotate towards a camera.

Usage
-------------
Apply a **CameraBillboard** script to an object and it will always rotate towards the selected camera.
